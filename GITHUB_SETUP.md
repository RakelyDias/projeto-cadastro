### 📤 Para Enviar ao GitHub:

**1. Criar repositório no GitHub:**

   - Descrição: "Sistema de cadastro de produtos com Node.js e MySQL"
   - Deixe em público (Public)
   - **NÃO** inicialize com README, .gitignore ou License (já temos)
   - Clique em "Create repository"

**2. Copie o comando que aparecerá (será algo como):**
   ```
   git remote add origin https://github.com/seu-usuario/cadastro-produtos.git
   git branch -M main
   git push -u origin main
   ```

### 🚀 Para Executar Localmente (Antes de usar):

**1. Configure o MySQL:**
   - Inicie o MySQL
   - Crie banco de dados `web_03mb`
   - Execute o SQL de `backend/database.sql`

**2. Inicie o backend:**
   ```bash
   cd backend
   npm start
   ```

**3. Abra o frontend:**
   - Use Live Server no VS Code ou
   - Abra `frontend/index.html` no navegador

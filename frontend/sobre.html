Sobre o Sistema

O que é este sistema?

O Sistema de Cadastro de Produtos é uma aplicação web desenvolvida para gerenciar de forma simples e eficiente o cadastro e visualização de produtos.

Tecnologias Utilizadas

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js com Express
  • Banco de Dados: MySQL

Funcionalidades Principais

  • ✅ Visualização de todos os produtos cadastrados
  • ✅ Cadastro de novos produtos com validação
  • ✅ Armazenamento seguro em banco de dados MySQL
  • ✅ Interface responsiva e intuitiva

Como Usar

  1. Navegue até Produtos para ver a lista de produtos
  2. Clique em Novo Produto para cadastrar um novo item
  3. Preencha o formulário com os dados do produto
  4. Clique em Cadastrar Produto para salvar
